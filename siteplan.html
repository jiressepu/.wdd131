Website Plan - Web Development Courses Platform

Site Name

Site Name: webacademy.com

Reason for the Name: "Webacademy.com" is simple and clear, reflecting the purpose of the site: to help users learn web creation, development, and programming. "Courses" indicates that this is a platform dedicated to training.

Suggested Domain: Webacademycourses.com (available as an example).

Site Purpose

The purpose of this site is to provide a comprehensive platform where users can discover, promote, and enroll in web development and programming courses. Users will be able to explore training programs based on their skill level, technology of interest (such as HTML, CSS, JavaScript, Python, etc.), and preferred course format (online courses, workshops, bootcamps, etc.).

Scenarios

  • How can I compare multiple web development courses to choose the best one for me?
  • What are the best courses for learning Python at a beginner level?
  • Where can I find in-person workshops for learning JavaScript?
  • How do I leave a review for a course I've taken?

Color Scheme

Primary Color: Green (#4CAF50) - Used for headings and call-to-action buttons.

Secondary Color: Orange (#FF5722) - Used for accents and important highlights.

These colors were chosen to create a strong contrast while maintaining a professional look.

Typography

Font 1: Arial - used for body text, descriptions, and menus.

Font 2: Roboto - used for headings and subheadings for a modern and clean look.

Wireframe

Below are wireframe sketches of the homepage layout for both desktop and mobile versions.

Wireframe - Desktop Version

  • Header: Logo, navigation menu (Home, Courses, Compare, Blog, Forum, About).
  • Main Section: Banner with an introductory message and a call-to-action button to explore courses.
  • Course List: Popular courses with filters for level, technology, and course type.
  • Review Section: Latest user reviews.
  • Footer: Legal info, social media links, and contact page link.

Wireframe - Mobile Version

  • Hamburger menu for navigation.
  • Compact intro section.
  • Swipeable course filters.
  • Horizontally scrollable review section.
  • Compact footer with essential links and social icons.

CSS

The project.css file applies the visual styles mentioned above. Key styling features include:

  • Headings styled with the Roboto font and bold weight.
  • Buttons and links styled with green and orange for clear visibility.
  • Responsive layout for desktop and mobile screens.
